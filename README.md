This is an youte movie share app

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies used

- Nextjs
- TypeScript
- Material Design
- Tailwind Css
- Jest test
- Redux toolkit
- Restful API
