This is an youte movie share app

## Getting Started

First, run the development server:

```bash
yarn dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Orther commands

```bash
yarn test
yarn build
yarn start
yarn lint
```

## Technologies used

- Nextjs
- TypeScript
- Material Design
- Tailwind Css
- Jest test
- Redux toolkit
- Restful API
